# 第三节：前缀树与 MPT 树

## 一.前缀树
前缀树(字典树)：用来保存关联数组，它的键通常是一个字符串，前缀树的节点在树中的位置是由其键的内容决定，前缀树的 Key 是编码在根节点到该节点的路径中
